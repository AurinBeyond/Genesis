# TOPIC-NAVIGATION-MAP
# Type: NAVIGATION AGENT
# Platform: AurinBeyond

---

## PURPOSE

Map each topic to its full user path.

User enters through a topic.
System routes them through the correct stages.

---

## NAVIGATION LOGIC

Each topic has its own path instance:

Entry → Awareness (Vaatlus) → Depth → Product → Onboarding → Continuation

No topic shares a path with another topic.

---

## ACTIVE TOPIC PATHS

### MONEY
- Entry: content about money patterns
- Awareness: how money conditioning works
- Depth: articles / videos on financial identity
- Product: mini-course or eBook on money freedom
- Onboarding: 24h protocol for money mindset reset
- Continuation: deeper financial system content

---

---

### IDENTITY
- Entry: content about self-perception

- Product: course or workbook on identity sovereignty

- Continuation: deeper self-architecture content

---

## ROUTING RULES

- User stays within chosen topic path
- Cross-topic links only offered after completion of current stage
- No forced redirection
- Navigation signals follow user action, not system pressure

---

## EXPANSION

New topics follow the same template.

Each new topic requires:
- Entry content
- Awareness page (Vaatlus)
- Depth content set
- Product assignment
- Onboarding sequence
- Continuation path
