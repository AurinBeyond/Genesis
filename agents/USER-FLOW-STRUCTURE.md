# USER FLOW STRUCTURE
# Platform: AurinBeyond
# Type: TOPIC-BASED USER JOURNEY SYSTEM

---

## GOAL

Move user through stages based on topic relevance:

first contact → awareness → content → product → continuation

No force. User chooses each step.

---

## STAGE 1: ENTRY POINT (FIRST CONTACT)

Where user enters:

- short video
- post
- social media content

Goal:

- attention
- recognition

---

## STAGE 2: AWARENESS PAGE (VAATLUS)

Replaces VSL.

Purpose:

- show the pattern
- create realization
- no selling pressure

Content structure:

- observation
- relatable example
- perception shift
- clarity

---

## STAGE 3: DEPTH CONTENT

User goes deeper:

- article
- video
- explanation

Goal:

- understanding

---

## STAGE 4: PRODUCT ENTRY (SOFT)

Only after clarity:

- mini-course
- eBook
- audio

No pressure.

Invitation only.

---

## STAGE 5: ONBOARDING

After entry:

- 24h protocol
- simple guidance
- slow integration

---

## STAGE 6: CONTINUATION

Long-term:

- deeper content
- next topics
- system expansion

---

## TOPIC-BASED FLOW

Each topic creates its own path.

### Example: Money

Entry → Awareness (money patterns) → Depth → Product

### Example: Relationships

Entry → Awareness → Depth → Product

Every topic follows the same structure.

Each path is independent.

---

## RULES

- no forcing movement
- user chooses next step
- clarity before product
- no aggressive funnel logic

---

## SUCCESS CONDITION

- user understands
- user relates
- user chooses to continue
- system feels natural
