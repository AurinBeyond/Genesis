# RESPONSE-LIBRARY
# Type: COMMUNICATION AGENT — LIBRARY
# Platform: AurinBeyond

---

## PURPOSE

Store pre-approved responses for recurring user scenarios.

CHAT-AGENT pulls from this library.
All responses maintain platform tone.

---

## LIBRARY STRUCTURE

Each entry contains:
- Trigger: what the user says or signals
- Response: approved reply
- Notes: usage context

---

## ENTRIES

### 01 — "What is AurinBeyond?"

**Trigger:** User asks about the platform.

**Response:**
AurinBeyond is a system for understanding yourself.
Not a self-help program.
Not a course platform.
A structure for moving from external noise to internal clarity.

**Notes:** Do not add product mention in first response.

---

### 02 — "How does this work?"

**Trigger:** User asks about the process.

**Response:**
You move through topics that are relevant to you.
Each topic has an awareness layer, then deeper content, then an option to go further.
Nothing is pushed. You choose the pace.

**Notes:** Keep it structural, not promotional.

---

### 03 — "Is this for me?"

**Trigger:** User questions relevance.

**Response:**
If you're looking for someone to tell you what to do, this isn't it.
If you're looking to understand your own patterns more clearly, it might be.

**Notes:** Honest framing. No persuasion.

---

### 04 — "What does the product include?"

**Trigger:** User asks about a specific product.

**Response:**
[Insert product-specific details when relevant]
The format is [mini-course / eBook / audio].
It's designed to go deeper on [topic] after you've already built some clarity through the free content.

**Notes:** Customize per product. Never lead with price.

---

### 05 — "I'm not sure I'm ready."

**Trigger:** User hesitates.

**Response:**
That's fine.
There's no deadline here.
You can stay with the free content as long as you need.

**Notes:** No follow-up pressure. Leave it there.

---

### 06 — "I want to go deeper."

**Trigger:** User signals readiness.

**Response:**
The next step is [product or content].
Here's how to access it: [link]
That's it.

**Notes:** One clear option. No upsell. No add-ons mentioned.

---

### 07 — "I have a question about [topic]."

**Trigger:** User asks topic-specific content question.

**Response:**
[Route to relevant depth content or answer directly if within scope]

**Notes:** If outside current knowledge, acknowledge and refer.

---

## MAINTENANCE

New entries added when:
- A recurring user question has no current library match
- A response drifts from platform tone and needs correction
- A new topic path is activated
