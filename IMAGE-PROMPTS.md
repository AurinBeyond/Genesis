## USAGE PRINCIPLE

These prompts are designed for AI image generation tools (Midjourney, DALL·E, Stable Diffusion, Flux, etc.).

**Style rules across all images:**
- Cinematic. Symbolic. Non-generic.
- Near-black backgrounds (`#050505`)
- Soft light sources — gold/white, never harsh
- No humans with faces (silhouettes only if human form is used)
- No text in images
- No stock-photo aesthetics
- 16:9 ratio for backgrounds, 3:2 for featured images

## USAGE NOTES

| Setting | Value |
|---------|-------|
| Aspect ratio | 16:9 (backgrounds), 3:2 (featured) |
| Resolution | Minimum 1920×1080 for backgrounds |
| Format | PNG or WebP preferred |
| File size | Compress to under 300KB for web use |
| Alt text | Descriptive, not decorative |

**Never use:**
- Generated images with faces or identifiable people
- Images that look AI-generated in obvious ways (uncanny, plastic)
- Bright, colorful, or high-saturation images
- Clip art, icons, illustration vectors styled as photos
