## PURPOSE

Define all product types used in the Pure Soul Life system.

Every product must belong to exactly ONE type. No exceptions. No overlap.

---

## TYPE 1 — ENTRY PRODUCT

**Role:** First point of transaction.

| Attribute | Value |
|-----------|-------|
| Complexity | Low |
| Price range | Low (49–199 NOK) |
| Time to consume | Short (15–30 min) |
| Time to create | Fast (1–3 days) |
| Purpose | Demonstrate value, reduce buyer friction, test delivery |

**Formats allowed:**
- PDF guide (mini-guide, checklist, protocol)
- Short audio (single recording, under 20 min)
- Single-page system (structured text)

**Rules:**
- Solves ONE specific problem
- Actionable immediately after purchase
- No prerequisites
- No multi-session content
- Delivered automatically via Gumroad

**Example:** *The Source Within — A Guide to Internal Authority* (PDF, 149 NOK)

---

## TYPE 2 — CORE PRODUCT

**Role:** Primary revenue product. The main system.

| Attribute | Value |
|-----------|-------|
| Complexity | Medium–High |
| Price range | Mid (499–1,999 NOK) |
| Time to consume | Multi-day (5–10 days) |
| Time to create | Structured (1–4 weeks) |
| Purpose | Deliver full transformation. Build authority. Main revenue driver. |

**Formats allowed:**
- Multi-day course (sequential modules)
- Protocol system (structured, phased)
- Step-by-step training program

**Rules:**
- Divided into sequential modules (see `COURSE-001.md`)
- Requires completion of Module N before Module N+1
- Delivered via email-based module unlock system
- Includes exercises and optional reflections per module
- Entry Product is NOT required but may be referenced

**Example:** *The Academy — Internal Authority Program* (7-module course)

---

## TYPE 3 — PREMIUM PRODUCT

**Role:** High-touch, deep-work engagement at premium price.

| Attribute | Value |
|-----------|-------|
| Complexity | High |
| Price range | High (5,000–20,000 NOK) |
| Time to consume | Extended (ongoing or session-based) |
| Time to create | Depends on personal availability |
| Purpose | Personal transformation with direct support. Maximum value delivery. |

**Formats allowed:**
- 1:1 session (single or package)
- Direct consultation + materials
- Advanced personal program with support

**Rules:**
- Not publicly listed as a default CTA on offer pages
- Accessed by referral, direct contact, or invitation
- Requires scheduling — not delivered automatically
- Price negotiated or listed privately
- Requires Core Product to exist first (product authority must be established)

**Example:** *1:1 Internal Authority Session* (60 min, direct booking)

---

## TYPE 4 — SUPPORT PRODUCTS

**Role:** Small add-ons that extend or deepen existing products.

| Attribute | Value |
|-----------|-------|
| Complexity | Very Low |
| Price range | Very Low (0–149 NOK) or bundled |
| Time to consume | Very short (5–15 min) |
| Time to create | Fast |
| Purpose | Add value without adding complexity. Upsell or bundle. |

**Formats allowed:**
- Templates (fillable documents)
- Exercise packs
- Reference tools
- Bonus audio or mini-modules
- Reflection prompts

**Rules:**
- Always attached to or extending a primary product (Type 1, 2, or 3)
- Never sold as standalone core offer
- Can be included as bonuses or low-cost upsells
- Must reference the parent product they support

**Example:** *Daily Reflection Protocol* (PDF template, included with Core Product)

## PRODUCT LADDER

```
TYPE 1 (Entry)     →  TYPE 2 (Core)     →  TYPE 3 (Premium)
    |                      |                      |
  Low cost             Mid price             High price
  Low friction         Main system         Personal depth
  First purchase       Core revenue        Maximum value

                TYPE 4 (Support) — attaches to any of the above
```

---

## ASSIGNMENT RULE

Before any product is created:

1. Assign it to one of the four types above
2. Confirm it does not duplicate an existing product of the same type
3. Confirm it fits the build sequence in `PRODUCT-ROADMAP.md`

No product is created without a type assignment.
